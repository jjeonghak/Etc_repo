## 함수 기반 인덱스
일반적인 인덱스는 칼럼의 값 일부 또는 전체에 대해서만 인덱스 생성 허용  
8.0 버전부터 칼럼의 값을 변형해서 만들어진 값에 대한 인덱스 지원  
- 가상 칼럼을 이용한 인덱스  
- 함수를 이용한 인덱스  

함수 기반 인덱스는 인덱싱할 값을 계산하는 과정의 차이만 존재하고 실제로는 B-Tree 인덱스와 동일
